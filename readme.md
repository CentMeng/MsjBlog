
# MsjBlog
#### 介绍
`Msj Blog`是基于SpringBoot2搭建的个人博客系统.
目前master分支的是基于SpringBoot2和Vue的前后端分离版本.`thymeleaf-version`是基于SpringBoot2的一体机代码.建议不熟悉Vue的可以切到这个版本使用.根据DimpleBlog改造。
